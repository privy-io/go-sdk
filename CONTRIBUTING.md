# Contributing

## Prerequisites

- Go 1.23+

## Setup

```sh
./scripts/bootstrap
```

This installs Go dependencies and optionally installs Homebrew packages on macOS.

## Code generation & releases

This SDK is generated by [Stainless](https://www.stainlessapi.com/).
Releases are managed automatically via
[release-please](https://github.com/googleapis/release-please).
All pull requests should be opened against the `next` branch, not `main`.

## Code structure

Most of the SDK is generated code. The following locations are safe to edit by hand:

- `privy_`-prefixed files (e.g. `privy_client.go`, `privy_user_service.go`)
- The `authorization/` package

Manual edits persist across regeneration but may cause merge conflicts with changes from the generator.

## Development workflow

### End-to-end tests

End-to-end tests run against the staging environment. Create a `.env` file with `TEST_APP_SECRET` set, then run:

```sh
go test -v ./e2e/...
```

### Lint & format

```sh
./scripts/lint
./scripts/format
```

Formatting uses the standard `gofmt` formatter.

## Local development

To use a local version of this library from source in another project, add a replace directive to your `go.mod`:

```sh
go mod edit -replace github.com/privy-io/go-sdk=/path/to/go-sdk
```
